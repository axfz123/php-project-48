## Requirements

- PHP >= 8.0
- Composer

## Setup

```
git clone https://github.com/axfz123/php-project-48.git
cd php-project-48
make install
```

## How to use

```
Usage:
gendiff (-h|--help)
gendiff (-v|--version)
gendiff [--format <fmt>] <firstFile> <secondFile>

Options:
-h --help                     Show this screen
-v --version                  Show version
--format <fmt>                Report format [default: stylish]
```
